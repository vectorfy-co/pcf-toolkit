# Platform Action

`platform-action` defines platform-level action behavior for your control.

## platform-action fields

| Field | Type | Description |
| --- | --- | --- |
| `action-type` | `afterPageLoad` | Action type. |

## Example

```yaml
platform-action:
  action-type: afterPageLoad
```
