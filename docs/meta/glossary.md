# Glossary

**ControlManifest.Input.xml**: The XML manifest consumed by Power Apps for PCF controls.

**PCF (Power Apps Component Framework)**: Framework for building custom components for Power Apps and Dynamics 365.

**Webresource**: A file (JS/CSS/HTML) packaged and served by Power Apps.

**PAC CLI**: Microsoft Power Apps Command Line Interface for managing environments.

**mitmproxy**: A local proxy tool that can intercept HTTPS traffic.

**Pydantic**: Python library used for data validation and typing.

**Deterministic XML**: XML output that is stable and consistently ordered across runs.

**Schema Snapshot**: A machine-readable reference extracted from official docs used to generate models.
