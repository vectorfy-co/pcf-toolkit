# Keyboard Shortcuts

These shortcuts are baked into the docs UI for faster navigation.

| Shortcut | Action |
| --- | --- |
| `Ctrl+K` / `Cmd+K` | Focus search input |
| `g` then `h` | Go to Home |
| `g` then `s` | Go to Quick Start |

Tip: use the sidebar to jump between major sections.
